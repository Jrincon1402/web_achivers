# Routes

Esta carpeta está reservada para rutas adicionales si decides migrar a Remix en el futuro.

En Next.js 13+, las rutas se manejan mediante el sistema de file-based routing
en la carpeta `app/` (por ejemplo: `app/about/page.tsx`).

Si en el futuro migras a Remix, puedes mover las rutas aquí siguiendo
la estructura de Remix file-based routing.
